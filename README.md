# NumPy Operations Project

## Introduction

This project is a menu-driven Python application developed using the NumPy library. It allows users to create an array and perform different mathematical operations through an interactive menu. The project is useful for understanding the basic implementation of NumPy arrays.

---

## Project Goal

The main goal of this project is to practice array manipulation and learn how NumPy simplifies numerical calculations in Python.

---

## Operations Included

- Create an integer array
- Increase all elements by a given value
- Multiply all elements with a user-defined number
- Arrange elements in ascending order
- Display the total sum
- Display the average of the array
- Exit the application safely

---

## Tools Used

- Python Programming Language
- NumPy Library
- Command Prompt / VS Code

---

## Installation

If NumPy is not installed, use:

```bash
pip install numpy
```

---

## Working Procedure

1. Launch the program.
2. Select **Create Array** from the menu.
3. Enter the array size and elements.
4. Choose any operation from the menu.
5. The program displays the result instantly.
6. Repeat the process until you select Exit.

---

## Error Handling

The program checks whether an array has been created before performing any operation. If not, it displays a message asking the user to create the array first.
